# 线程池
线程能够很好地提升进程执行速度和提高资源利用。而仅仅在需要一个线程的时候才去创建的话，在某些情况下就会带来问题：如果并发的线程数量众多，并且每个线程都是执行很短的时间就结束了，这种场景下，频繁创建/销毁线程就会浪费很多时间，在很大程度上降低系统效率。
此时，一个可行的方案就是，能否对线程进行复用——执行完一个任务后，委派其他任务给它继续执行，而不是将其销毁。java 中称之为线程池技术。
